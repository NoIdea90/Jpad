Is a cute lil macropad with 9 keys, 1 rotary encoder, and 1 oled screen.

# challenge
Jpad was made with exactly 0 experience in anything, coding, 3d modeling, schematics, pcb, none.
so naturally that led to me remaking the pcb about a million times from scratch, also the case

# features
An awesome design 
An OLED display that is currently very boring but i have some plans for it
9 cherry mx switches (wired in a matrix btw)
One EC11 rotary encoder with a switch

# BOM 
9x Cherry MX switches
9x through-hole 1N4148 Diodes
1x seeed studios xiao RP2040
1x 0.91 inch OLED display
4x M3 heatset inserts
4x M3 screws
